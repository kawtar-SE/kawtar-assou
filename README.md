//Features//

*Search movies by title

*View movie details ( rating, release date)

*Browse Top Rated & Popular movies

*Fast performance (Vite + React)


//Tech Stack//

React (Vite)

Tailwind CSS (optional)

OMDB API

React Router

//Installation & Setup//

Clone the project:

git clone https://github.com/kawtar-SE/movie-search-app.git
cd movie-search-app
